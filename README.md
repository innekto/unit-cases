# Runner Backend

## Description

Created with [Nest](https://github.com/nestjs/nest) framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Lint

```bash
# file system linter
$ npm run lint:fs

# JS/TS linter
$ npm run lint:js

# types linter
$ npm run lint:type

# editor linter
$ npm run lint:editor

# prettify linter
$ npm run lint:format
```
